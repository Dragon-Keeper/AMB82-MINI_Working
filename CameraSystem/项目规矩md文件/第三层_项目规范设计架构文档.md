## 二、Arduino/C++编码规范

### 2.1 基本代码结构

Arduino程序的基本结构包括setup()和loop()两个函数，代码应该清晰组织。

```cpp
/*
 * 项目名称：智能温控器
 * 功能描述：基于ESP32的温度控制系统
 * 作者：[作者名]
 * 日期：2024-01-15
 * 版本：v1.0.0
 * 
 * 硬件连接：
 * - OLED: SDA=GPIO4, SCL=GPIO5
 * - 编码器: CLK=GPIO18, DT=GPIO19, SW=GPIO21
 * - 温度传感器: DS18B20=GPIO34
 */

// ==================== 头文件引用 ====================
#include <Wire.h>
#include <Adafruit_GFX.h>
#include <Adafruit_SSD1306.h>
#include <RTClib.h>

// ==================== 引脚定义 ====================
#define PIN_SDA 4
#define PIN_SCL 5
#define PIN_ENCODER_CLK 18
#define PIN_ENCODER_DT 19
#define PIN_ENCODER_SW 21
#define PIN_TEMP_SENSOR 34

// ==================== 常量定义 ====================
#define SCREEN_WIDTH 128
#define SCREEN_HEIGHT 64
#define MAX_TEMPERATURE 100
#define MIN_TEMPERATURE 0

// ==================== 全局变量 ====================
float g_current_temp = 0.0;    // 当前温度
int g_target_temp = 25;        // 目标温度
bool g_heater_on = false;      // 加热器状态

// ==================== 对象实例 ====================
Adafruit_SSD1306 display(SCREEN_WIDTH, SCREEN_HEIGHT, &Wire, -1);
RTC_DS3231 rtc;

// ==================== 函数声明 ====================
void init_display();
void init_rtc();
void read_temperature();
void update_display();
void handle_encoder();

// ==================== 初始化函数 ====================
void setup() {
  // 初始化串口，用于调试
  Serial.begin(115200);
  Serial.println("系统启动中...");
  
  // 初始化各模块
  init_display();
  init_rtc();
  
  // 配置引脚模式
  pinMode(PIN_ENCODER_CLK, INPUT);
  pinMode(PIN_ENCODER_DT, INPUT);
  pinMode(PIN_ENCODER_SW, INPUT_PULLUP);
  
  Serial.println("系统启动完成！");
}

// ==================== 主循环 ====================
void loop() {
  // 读取温度
  read_temperature();
  
  // 处理编码器输入
  handle_encoder();
  
  // 更新显示
  update_display();
  
  // 温度控制逻辑
  if (g_current_temp < g_target_temp) {
    g_heater_on = true;
  } else {
    g_heater_on = false;
  }
  
  // 延时，避免刷新过快
  delay(100);
}

// ==================== 功能函数实现 ====================
/**
 * @brief 初始化OLED显示屏
 */
void init_display() {
  Wire.begin(PIN_SDA, PIN_SCL);
  
  if (!display.begin(SSD1306_SWITCHCAPVCC, 0x3C)) {
    Serial.println("OLED初始化失败！");
    while (1);  // 停止运行
  }
  
  display.clearDisplay();
  display.setTextSize(1);
  display.setTextColor(WHITE);
  display.setCursor(0, 0);
  display.println("系统启动中...");
  display.display();
  
  Serial.println("OLED初始化成功");
}

/**
 * @brief 初始化RTC模块
 */
void init_rtc() {
  if (!rtc.begin()) {
    Serial.println("RTC初始化失败！");
    while (1);
  }
  
  // 如果RTC时间不正确，设置为编译时间
  if (rtc.lostPower()) {
    rtc.adjust(DateTime(F(__DATE__), F(__TIME__)));
    Serial.println("RTC时间已同步");
  }
  
  Serial.println("RTC初始化成功");
}
```

#### 3.2.2 旋转编码器模块

```markdown
## 旋转编码器模块 (KY-040)

### 基本信息
- 型号：KY-040
- 类型：增量式编码器
- 每圈脉冲：20
- 工作电压：3.3V-5V

### 引脚定义
| 引脚 | 功能 | 说明 |
|------|------|------|
| VCC | 电源 | 3.3V-5V |
| GND | 地 | 公共地 |
| CLK | 时钟 | 输出A相 |
| DT | 数据 | 输出B相 |
| SW | 按键 | 按下为低电平 |

### 工作原理
- 顺时针旋转：CLK先变高，DT后变高
- 逆时针旋转：DT先变高，CLK后变高
- 按下按键：SW变为低电平

### 使用示例
```cpp
#define PIN_CLK 18
#define PIN_DT 19
#define PIN_SW 21

int last_clk = HIGH;
int encoder_count = 0;

void setup() {
  pinMode(PIN_CLK, INPUT);
  pinMode(PIN_DT, INPUT);
  pinMode(PIN_SW, INPUT_PULLUP);
  last_clk = digitalRead(PIN_CLK);
}

void loop() {
  int clk = digitalRead(PIN_CLK);
  
  if (clk != last_clk && clk == HIGH) {
    if (digitalRead(PIN_DT) != clk) {
      encoder_count--;  // 逆时针
    } else {
      encoder_count++;  // 顺时针
    }
    Serial.println(encoder_count);
  }
  
  last_clk = clk;
  
  // 检测按键
  if (digitalRead(PIN_SW) == LOW) {
    Serial.println("按键按下");
    delay(200);  // 消抖
  }
}
```
```

## 六、常见问题

### Q1: 代码注释应该写多少？

**A**: 
- 每个函数必须有功能说明
- 复杂逻辑需要行内注释
- 新手项目建议注释详细一些
- 注释应该解释「为什么」，而非「是什么」

### Q2: 如何管理多个硬件版本？

**A**: 
- 使用条件编译区分版本
- 在文档中记录版本差异
- 使用版本号管理配置

### Q3: 如何处理硬件异常？

**A**: 
- 添加超时检测
- 使用看门狗定时器
- 记录错误日志
- 提供恢复机制
